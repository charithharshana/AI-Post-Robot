## üöÄ Quick Start

### Installation
1. **Download**: Clone or download this repository
   ```bash
   git clone https://github.com/charithharshana/AI-Post-Robot.git
   ```
2. **Load Extension**:
   - Open Chrome/Edge and navigate to `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked" and select the extension directory
3. **Configure APIs**: Set up your AI Post Robot and Gemini API keys in settings

### Initial Setup
1. **AI Post Robot API**: Sign up at [AI Post Robot](https://app.aipostrobot.com), get API key, configure in settings
2. **Google Gemini API** (Optional): Get key from [Google AI Studio](https://aistudio.google.com/), configure for AI features
3. **Start Using**: Visit Facebook/Pinterest for auto-capture, or use Advanced Scheduler for manual content management

### Basic Usage
- **Auto-Capture**: Content automatically captured while browsing
- **Manual Capture**: Select text or right-click images to save
- **AI Image+**: Click "AI Image+" in Advanced Scheduler to create/edit images with AI
- **Schedule Posts**: Use popup or Advanced Scheduler for bulk scheduling
